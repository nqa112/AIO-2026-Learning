# Image Similarity Search

Use distance metrics such as L1, L2, Cosine to find the most similar image(s) to the query image.

## Project Structure

- `app.py`: Builds the Streamlit interface.
- `main.py`: Contains the main image processing and search logic.
- `metrics.py`: Contains the distance metrics.
- `Data/`: Folder containing default query image and images used as the search database.
